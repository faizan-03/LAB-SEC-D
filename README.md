## Learning Goals

- Apply classes and objects in practical exercises.
- Practice constructors and destructors.
- Understand encapsulation and method design.
- Build confidence with input/output and program execution.

Use any C++ compiler (GCC/MinGW recommended).

```bash

./lab3
```
